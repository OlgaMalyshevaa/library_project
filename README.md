# Library project

## Запуск проекта
### 1) склонируйте репозиторий
```bash
git clone https://github.com/username/library_project.git
```
### 2) установите зависимости 
```bash
go mod tidy
```

### 3) создайте файл .env

### 4) запустите проект
```bash
docker-compose up --build
```

## Доступ к Swagger:
Swagger UI доступен по адресу:
```
http://localhost:8080/swagger/index.html
```
